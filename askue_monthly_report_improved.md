## 6. Техническая информация

### 6.1 Расчетные параметры

**Коэффициент использования мощности:**
```
K_исп = (P_средняя / P_максимальная) × 100% = (167.8 / 218.5) × 100% = 76.8%
```
*Интерпретация: График нагрузки равномерный, возможна оптимизация пиковых значений.*

**Удельное потребление энергии на единицу продукции:**
```
e_уд = E_акт / Q_продукции = 1,245,300 кВт·ч / [объем производства] кВт·ч/ед.
```
